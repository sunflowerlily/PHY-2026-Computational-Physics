# PHY-2026-Computational-Physics

本仓库包含这门 18 周课程的完整教学资源，旨在连接物理理论推导与计算机数值模拟。课程内容涵盖误差分析、线性代数、ODEs/PDEs 数值解法、蒙特卡洛模拟及 AI for Science 前沿应用。

## Course Structure

本课程为期 18 周，每周包含理论课（lecture）和实验课（lab）两部分内容。

```
weeks/
├── week01/
│   ├── lecture/
│   └── lab/
├── week02/
│   ├── lecture/
│   └── lab/
...
└── week18/
    ├── lecture/
    └── lab/
```

## Grading Policy

本课程采用独特的"理论+实验"双轨独立考核体系：

### Theory Track (理论课考核)
侧重于算法原理、稳定性证明与数学推导。

- **Quizzes (课堂测验)**: 20%
- **Homework (作业)**: 30%
- **Midterm Exam (期中考试)**: 20%
- **Final Exam (期末考试)**: 30%

### Lab Track (实验课考核)
侧重于 Python 编程实现、Debug 能力与数据可视化。

- **Weekly Labs (每周实验)**: 40%
- **Code Checks (代码检查)**: 10%
- **Mini-Project (小项目)**: 20%
- **Final Project (期末项目)**: 30%
