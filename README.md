# PHY-2026-Computational-Physics
本仓库包含这门 18 周课程的完整教学资源，旨在连接物理理论推导与计算机数值模拟。课程内容涵盖误差分析、线性代数、ODEs/PDEs 数值解法、蒙特卡洛模拟及 AI for Science 前沿应用。  本课程采用独特的“理论+实验”双轨独立考核体系：  理论课：侧重于算法原理、稳定性证明与数学推导。实验课：侧重于 Python 编程实现、Debug 能力与数据可视化。

## 📊 评分标准 (Grading Scheme)

本课程总成绩由 **4 个核心部分** 组成，理论与实验能力并重。

| 组成部分 (Component) | 占比 (Weight) | 形式 (Format) | 说明 (Description) |
| :--- | :--- | :--- | :--- |
| **1. 平时实验 (Weekly Labs)** | **30%** | `Jupyter Notebook` | 每周实验课完成。重点考察代码实现能力、Debug 能力及可视化分析。 |
| **2. 理论作业 (Theory HW)** | **20%** | `PDF / Markdown` | 双周发布一次。重点考察算法误差分析、物理建模推导 (不涉及大量编程)。 |
| **3. 期中考核 (Midterm)** | **20%** | `闭卷笔试` 或 `上机` | 第 9 周进行。考察核心算法原理（如泰勒展开、误差传递）及手写伪代码能力。 |
| **4. 期末项目 (Final Project)** | **30%** | `Code + Report + Pre` | 第 15-18 周。分组完成一个复杂的物理系统模拟（如三体问题、伊辛模型）。 |

> ⚠️ **及格红线**：若“平时实验”或“期末项目”任意一项得分低于 50%，总成绩将自动记为不及格。我们强调动手能力。

## 🗓️ 教学进度 (Schedule)

每周包含 **[L] 理论课** (原理与推导) 和 **[E] 实验课** (编程与验证)。

### Phase 1: 基础与工具 (Weeks 1-3)
| Week | [L] 理论主题 (Theory) | [E] 实验任务 (Lab) |
| :--- | :--- | :--- |
| **01** | **计算物理新范式**<br>误差分析、IEEE 754、截断 vs 舍入 | **Lab 01: 捕捉幽灵误差**<br>验证机器精度，绘制微分误差 V 型曲线 |
| **02** | **线性代数与量子力学**<br>矩阵运算、本征值问题、基底变换 | **Lab 02: 量子势阱求解**<br>使用 `numpy.linalg` 求解一维势阱能级 |
| **03** | **数据处理与插值**<br>最小二乘法、样条插值、FFT 原理 | **Lab 03: 实验数据拟合**<br>处理带噪声的物理实验数据，频谱分析 |

### Phase 2: 经典动力学 ODEs (Weeks 4-7)
| Week | [L] 理论主题 (Theory) | [E] 实验任务 (Lab) |
| :--- | :--- | :--- |
| **04** | **初值问题 (IVP)**<br>欧拉法、RK4 方法推导与稳定性分析 | **Lab 04: 炮弹与空气阻力**<br>对比 Euler 与 RK4 精度，模拟抛体运动 |
| **05** | **混沌与非线性**<br>相空间、庞加莱截面、李雅普诺夫指数 | **Lab 05: 蝴蝶效应**<br>模拟 Lorenz 吸引子，计算对初值的敏感度 |
| **06** | **天体与哈密顿力学**<br>辛算法 (Symplectic)、能量守恒特性 | **Lab 06: 三体问题**<br>使用 Verlet 算法模拟行星轨道，验证能量守恒 |
| **07** | **分子动力学 (MD)**<br>多体相互作用、Lennard-Jones 势 | **Lab 07: 氩气模拟**<br>模拟 N 个粒子的运动与碰撞 (Box Simulation) |

### Phase 3: 场与波 PDEs (Weeks 8-11)
| Week | [L] 理论主题 (Theory) | [E] 实验任务 (Lab) |
| :--- | :--- | :--- |
| **08** | **边值问题 (BVP)**<br>打靶法 (Shooting)、有限差分法 | **Lab 08: 量子隧穿**<br>求解定态薛定谔方程的边值问题 |
| **09** | **期中复习与考核 (Midterm)** | **Midterm Exam** (无实验课) |
| **10** | **椭圆型方程 (静场)**<br>拉普拉斯方程、松弛法 (Jacobi/SOR) | **Lab 09: 电势分布可视化**<br>计算复杂边界条件下的电场分布 |
| **11** | **抛物型与双曲型方程**<br>热传导方程、波动方程、稳定性条件 | **Lab 10: 热扩散与波传播**<br>模拟金属板热传导及琴弦振动 |

### Phase 4: 随机过程与统计 (Weeks 12-14)
| Week | [L] 理论主题 (Theory) | [E] 实验任务 (Lab) |
| :--- | :--- | :--- |
| **12** | **随机数与随机游走**<br>伪随机数生成器、中心极限定理 | **Lab 11: 布朗运动**<br>模拟 2D 随机游走，验证扩散系数 $D$ |
| **13** | **蒙特卡洛方法 (MC)**<br>高维积分、重要性采样 | **Lab 12: 计算 $\pi$ 与高维体积**<br>使用 MC 方法求解无法解析积出的积分 |
| **14** | **统计物理模型**<br>伊辛模型 (Ising Model)、Metropolis 算法 | **Lab 13: 磁相变模拟**<br>模拟铁磁-顺磁相变，观察居里温度 |

### Phase 5: 前沿与项目 (Weeks 15-18)
| Week | [L] 理论主题 (Theory) | [E] 实验任务 (Lab) |
| :--- | :--- | :--- |
| **15** | **AI for Science (AI4S)**<br>神经网络求解微分方程 (PINNs) 简介 | **Lab 14: 简单的 PINN**<br>用 PyTorch 训练一个网络拟合简谐振动 |
| **16** | **项目启动与选题**<br>如何管理一个代码项目、Git 协作流 | **Project: 组队与开题报告**<br>确定课题，建立 GitHub 仓库 |
| **17** | **项目工坊 (Workshop)**<br>助教/老师 驻场答疑与 Code Review | **Project: 核心代码攻关**<br>完成模拟核心逻辑，生成初步数据 |
| **18** | **最终展示 (Presentation)** | **Project: 成果汇报**<br>PPT 演讲 + 现场代码演示 |

## 🛠️ 提交指南 (Submission)

### 1. 实验报告 (Labs)
- 截止时间：每实验课后 1 周。
- 提交格式：`.ipynb` 文件，需包含完整的代码、运行结果图表以及 Markdown 格式的分析文字。
- 命名规范：`LabXX_StudentName_ID.ipynb`

### 2. 理论作业 (Theory HW)
- 提交格式：PDF (推荐 LaTeX 编写，也可手写扫描)。
- 提交位置：上传至课程系统的 `Homework` 文件夹。

## 🤖 AI 辅助政策 (AI Policy)

我们拥抱工具，但拒绝依赖。

1.  **允许**：使用 ChatGPT/Copilot 解释报错、生成代码片段、优化绘图风格。
2.  **禁止**：直接复制粘贴 AI 生成的物理推导过程而不加验证。
3.  **要求**：如果你的代码由 AI 辅助生成，请在 Notebook 开头添加 `## AI Acknowledgement` 章节，简述你如何使用了 AI，以及你如何验证了它是对的。
