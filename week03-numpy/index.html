第 2 周：Numpy 场论与高维时空（向量化思维）

本周核心：从“单粒子循环”进化到“全空间场论”。在 Python 中，循环是性能的黑洞，向量化是唯一的救赎。

1. 物理学家的编程世界观 (The Physicist’s Mindset)¶

1.1 粒子视角 vs. 场论视角¶

在计算物理中，我们有两种看待数据的方式：

  • 粒子视角 (The Particle View)：

    • 关注单个数据点 xix_ixi​。

    • 对应编程结构：for 循环。

    • 物理类比：牛顿力学中的质点轨迹追踪。

    • Python 现状：极慢。解释器需要对每个粒子进行类型检查和调度。

  • 场论视角 (The Field View)：

    • 关注整个空间分布 Ψ(x)\Psi(x)Ψ(x)。

    • 对应编程结构：Numpy 向量化 (Vectorization)。

    • 物理类比：电磁场或流体力学，整个场作为一个整体同时演化。

    • Python 现状：极快。底层调用 C/Fortran 的 BLAS/LAPACK 库进行并行计算。

🚀 教学目标：本周我们将学会“拒绝循环 (No Loops)”，训练自己看到数组时，不再看到一个个数字，而是看到一个完整的向量、矩阵或张量。

2. 核心武器：广播机制 (Broadcasting) —— 维度的张量积¶

广播是 Numpy 最难理解但也最强大的特性。

2.1 物理翻译：张量积与平行宇宙¶

假设我们要计算两组粒子 {r⃗i}\{ \vec{r}_i \}{ri​} 和 {R⃗j}\{ \vec{R}_j \}{Rj​} 之间的相互作用势（例如引力势）。

Vij=−GmiMj∣r⃗i−R⃗j∣V_{ij} = -G \frac{m_i M_j}{|\vec{r}_i - \vec{R}_j|}Vij​=−G∣ri​−Rj​∣mi​Mj​​
(1)

  • 传统思维：双重循环，遍历每一个 iii 和 jjj。

  • 广播思维：构建张量积空间 (Tensor Product Space)。

我们可以把数组看作存在于不同的“维度”或“平行宇宙”中：

  • 粒子群 A：形状 (N, 1)。它在第 0 维有定义，在第 1 维是“真空”（但可以延伸）。

  • 粒子群 B：形状 (1, M)。它在第 0 维是“真空”，在第 1 维有定义。

当它们相减时，Numpy 会自动把它们“延伸”到对方的维度，形成一个 (N, M) 的相互作用矩阵。

2.2 演示代码：计算距离矩阵¶

import numpy as np

# 粒子位置 (N=3, M=2)
r = np.array([0, 1, 2])       # Shape: (3,)
R = np.array([10, 20])        # Shape: (2,)

# ❌ 错误做法：直接相减
# r - R  # ValueError: operands could not be broadcast together

# ✅ 正确做法：升维 (增加 Axis)
r_col = r[:, np.newaxis]      # Shape: (3, 1)  <- 像列向量
R_row = R[np.newaxis, :]      # Shape: (1, 2)  <- 像行向量

# 广播发生时刻
diff = r_col - R_row          # Shape: (3, 2)
# 结果矩阵 D_ij = r_i - R_j
# [[ -10,  -20],
#  [  -9,  -19],
#  [  -8,  -18]]

💡 物理直觉：这就像是量子力学中，通过直积 HA⊗HB\mathcal{H}_A \otimes \mathcal{H}_BHA​⊗HB​ 构建复合系统的希尔伯特空间。

3. 连续与离散：插值 (Interpolation)¶

实验数据是离散的采样点（Lattice），而物理定律通常定义在连续流形（Manifold）上。我们需要从离散恢复连续。

3.1 多项式插值 vs. 样条插值¶

  • 拉格朗日插值 (Lagrange Polynomial)：

    • 试图用一个高阶多项式穿过所有点。

    • 物理陷阱：龙格现象 (Runge’s Phenomenon)。在边界处，多项式会剧烈振荡。这就像是高能截断带来的非物理发散。

  • 三次样条 (Cubic Spline)：

    • 分段使用三次多项式，保证函数值、一阶导（速度）、二阶导（加速度/曲率）连续。

    • 物理本质：最小作用量原理。

    • 它模拟了一根经过所有数据点的弹性金属条（Spline），其弹性势能（弯曲能量）最小化：

      E=∫∣f′′(x)∣2dx→min⁡E = \int |f''(x)|^2 dx \rightarrow \minE=∫∣f′′(x)∣2dx→min
      (2)

3.2 什么时候用哪个？¶

  • 数据点少且平滑 -> 多项式。

  • 数据点多或变化剧烈 -> 样条插值 (Spline)（首选）。

4. 科学可视化：从数据到直觉¶

绘图不是为了“好看”，而是为了保留物理真实性。

4.1 关键原则：Aspect Ratio¶

当你绘制物理场（如电场线、流线、行星轨道）时，必须保证 xxx 轴和 yyy 轴的比例尺一致。

import matplotlib.pyplot as plt

fig, ax = plt.subplots()
ax.plot(x, y)

# ⚠️ 物理生死线
ax.set_aspect('equal') 
# 如果不加这行，你的圆形轨道会变成椭圆，导出的切线方向（速度）全是错的！

4.2 颜色的物理意义 (Colormap)¶

  • Sequential (如 viridis)：表示密度、温度、概率（从 0 到正无穷）。

  • Diverging (如 coolwarm, seismic)：表示有正有负的量（如电势、涡度），必须把 0 点对准颜色中心（白色）。
