## 🤖 3. AI 的角色：是副驾驶，不是导航员

## 🔄 4. 核心方法论：计算物理的闭环
[Image Prompt: 一个发光的五色圆环流程图，依次是：物理现象 -> 数学建模 -> 确定算法 -> 编程计算 -> 可视化分析 -> 回到物理]

### 解决问题的六步法
1.  **物理现象**：观察现实。
2.  **数学建模**：写出方程 (Model)。
3.  **确定算法**：离散化 (Algorithm)。
4.  **编程计算**：写代码 (Code)。
5.  **可视化**：画图 (Visualization)。
6.  **验证**：结果合理吗？(Validation)。

> **本课重点**：虽然完整的闭环包含了发现定律，但我们将**专注于第 3、4、5 步**——即如何把方程变成代码和图像。

## � 5. 核心概念：什么是算法？
[Image Prompt: 左边是牛顿手写的平滑微积分公式，右边是计算机屏幕上离散的像素块和阶梯状的函数图像，中间用等号连接]

### 物理定律的“离散替身”
*   **解析解 (God's View)**：连续、完美、瞬间知晓过去未来 ($\vec{F}=m\vec{a}$)。
*   **数值算法 (Mortal's View)**：离散、近似、一步步推演 ($\vec{x}_{t+1} = \vec{x}_t + \vec{v}_t \Delta t$)。
*   **代价**：用有限逼近无限，注定产生**误差**。
